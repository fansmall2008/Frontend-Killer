# Web GameList Oper

Welcome to the Web GameList Oper Wiki! This is a web application for managing game lists with support for multiple frontend templates.

## Features

- 🎮 **Game List Management** - Create, edit, and manage game entries
- 📥 **Import** - Import game data from various frontend templates (Pegasus, RetroBat, etc.)
- 📤 **Export** - Export game data to multiple frontend formats
- 🌐 **Internationalization** - Support for Chinese, English, and Japanese
- 🖼️ **Media Management** - Manage game images, videos, and manuals
